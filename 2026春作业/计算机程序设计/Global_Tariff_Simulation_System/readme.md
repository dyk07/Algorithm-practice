全球关税模拟系统 - 操作说明

1. 环境要求
   - Python 3.6 或更高版本
   - 无需额外安装依赖（仅使用标准库）

2. 运行方式
   - 将 所需要用到的所有文件 置于同一目录
   - 在终端（确保地址为当前目录）
      执行：python tariff_system.py，进入图形用户界面
      执行：python tariff_system.py --cli，进入命令行界面
   - 首次运行会自动创建 tariff_data.json 文件保存数据，该文件作为系统所有当前数据的储存文件

3. 操作指南
   - 如果使用命令行界面，程序启动后显示主菜单，根据提示输入数字选择功能
   - 所有输入均需按照提示进行，错误输入会得到相应提示
   - 国家代码不区分大小写，商品名称不区分大小写
   - 价格、过境费、税率均为非负数，支持小数
   - 路径输入示例：CN>JP>US（使用 > 分隔）

4. 功能列表
   (1) 国家管理：增删改查国家，删除时自动清理关联数据
   (2) 商品管理：增删商品，设置/取消生产价格
   (3) 关税管理：设置/查询关税，打印关税矩阵，撤销已进行的关税调整
   (4) 指定路径价格计算：输入路径，输出每步价格
   (5) 最优采购路径查询：自动搜索一定跳数内最便宜路径（默认4跳，可手动修改），输出路径和价格
   (6) 制造业回流分析：计算最小关税调整方案，可以选择是否应用于关税调整（可在GUI的“关税管理界面”或CLI的“系统功能”一键撤销）
   (7) 系统功能：数据导入与导出（与tariff_data.json文件格式一致），撤销
   (8) 退出程序

5. 注意事项
   - 制造业回流分析中，调整方案仅输出建议，应用前会询问确认
   - 当前的关税数据自动保存，下次启动自动加载
   - 若需重置系统，删除 tariff_data.json 文件即可
   - **特别提醒**：历史记录仅保存在运行时内存中，程序重启后将无法数据回滚
